Regional Module processing software

Installation (on windows)

Required packages:

  • Python 3.4 64 bit.
  • xlsxWriter, tested under version 0.7.3.
  • xlrd, tested under version 0.9.4. However, xlrd has some issues in processing comments. A fix patch is submitted to the maintainer, but has not beed integrated yet. Therefore, until the patch is integrated please use the xlrd 0.9.4 version in the following github repo.

All those packages could be installed using Pip, which could be installed by downloading the get-pip.py and run the following in the terminal where get-pip.py exist. (for more details see this)

python get-pip.py Once Pip is installed run the following in your terminal

pip install xlsxwriter pip install https://github.com/melmasri/xlrd.zip

Installation of the main package

download or clone this repository

  • To download: click on the download link on the left, or here.
  • To clone: in your terminal run git clone https://github.com/melmasri/RMS

Move the downloaded/cloned folder to your desired destination and then create a shortcut to RM_GUI.py.

To start double-click on the RM_GUI.py shortcut.
